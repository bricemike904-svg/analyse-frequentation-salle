# Analyse de la fréquentation d'une salle de sport

## Contexte
Le gérant d'une salle de sport souhaite mieux comprendre la fréquentation afin:
-d'identifier les jours et heures les plus fréquentés,
-de connaître les abonnements les plus rentables,
-d'optimiser les horaires et promotions.

## Objectifs
- Nettoyer un jeu de données contenant des errurs potentielles
- Analyser le nombre de clients par jour et par créneau horaire
- Calculer le revenu par type d'abonnement
- Fournir des recommandations stratégiques

## Outils
- Python
- Pandas
- Matplotlib

## Méthodologie
1. Chargement et inspection des données
2. Nettoyage et correction des types si nécessaire
3. Calcul du chiffre d'affaires (revenu)
4. Analyses statistiques :
	- Total de clients par jour
	- Total de revenu par type d'abonnement
	- Clients par créneau horaire
5. Visualisations graphiques
6. Interprétation et recommandations

## Résultats clés
- Le **samedi** est le jour le plus fréquenté
- Le **créneau 18h** est le plus chargé
- Les abonnements **annuels** génèrent le plus de revenu
- Les abonnements mensuels sont moins rentables mais nombreux

## Recommandations
- Prévoir plus de personnel et équipement les samedis et soirées
- Promouvoir les abonnements annuels pour augmenter le revenu
- Adapter les cours et offres spéciales aux créneaux les plus fréquentés.
